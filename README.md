# Naga Tourist Spot Recommendation System

## Description

This is a Flask-based web application that allows users to:

- Register and login
- Create posts (reviews)
- Edit and delete posts
- View other users' content

## Tech Stack

- Backend: Flask (Python)
- Database: SQLite / SQLAlchemy
- Frontend: HTML, CSS, JS (or Vue)

## Features

- User Authentication (Login/Register)
- CRUD Operations (Posts)
- User Profiles
- Follow System
